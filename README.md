# 雨夌站

这是一个基于 Material Design 3 的静态个人网页

## 项目结构

## 特点

- 采用 MD3 设计规范
- 暗黑主题配色方案
- 动态加载组件（页头、侧栏菜单、页脚）
- 无缝页面切换，无闪屏
- 响应式设计，适配各种设备

## 部署到 GitHub Pages

1. 将项目推送到 GitHub 仓库
2. 在仓库设置中，导航到 "Pages" 选项
3. 选择分支（通常为 main 或 master）
4. 设置根目录作为网站源
5. 点击保存，等待几分钟后即可访问

## 技术栈
