### 附录 A：术语表（Glossary）

本术语表按字母顺序排列，覆盖维基正文中出现的所有核心技术术语。

| 术语            | 英文全称                                        | 中文释义                                                  |
| :-------------- | :---------------------------------------------- | :-------------------------------------------------------- |
| **Carbon M**    | Carbon M Series                                 | Leaf Design 自研显示芯片系列，负责屏幕渲染与极限模式输出  |
| **CRL**         | Certificate Revocation List                     | 证书吊销列表，用于标记已失效的数字证书                    |
| **DMA**         | Direct Memory Access                            | 直接内存访问，允许硬件设备绕过 CPU 直接读写内存           |
| **eFuse**       | Electronic Fuse                                 | 电子熔丝，一次性可编程存储器，烧断后不可恢复              |
| **EL3**         | Exception Level 3                               | ARM 架构中的最高异常级别，Leaf OS 内核运行于此级别        |
| **FUSE_BL**     | Bootloader Fuse                                 | Bootloader 解锁熔断位，烧断后 PACT 功能永久禁用           |
| **FUSE_TAMPER** | Tamper Detection Fuse                           | 物理入侵探测熔断位，触发后根密钥零化                      |
| **GPIO**        | General Purpose Input/Output                    | 通用输入输出引脚，用于硬件级信号控制                      |
| **HMAC**        | Hash-based Message Authentication Code          | 基于哈希的消息认证码，用于验证消息完整性与真实性          |
| **HSM**         | Hardware Security Module                        | 硬件安全模块，用于密钥管理与加密运算的物理隔离设备        |
| **Hypercall**   | Hypervisor Call                                 | 虚拟机监控器调用，用于在虚拟机与宿主机之间通信            |
| **Hypervisor**  | Type-1 Hypervisor                               | 裸机型虚拟机监控器，直接运行于硬件之上，管理 Android 沙盒 |
| **I2S**         | Inter-IC Sound                                  | 集成电路间音频总线，用于传输数字音频数据                  |
| **I3C**         | Improved Inter-Integrated Circuit               | 改进型 I²C 总线，Leaf OS 私有安全总线的基础               |
| **IPC**         | Inter-Process Communication                     | 进程间通信，Leaf OS 基于能力模型的通信机制                |
| **ISA**         | Instruction Set Architecture                    | 指令集架构，Trdom 私有指令集，不兼容 ARM/x86              |
| **LBT**         | Leaf Boot Tool                                  | Leaf OS 恢复模式，类比 Android Recovery                   |
| **LDB**         | Leaf Debug Bridge                               | Leaf OS 调试桥，类比 Android ADB                          |
| **LFR**         | Low-Frequency Reconstruction                    | 低频轮廓重建，Trneuro 在低功耗模式下的轻量级修复模型      |
| **Merkle Tree** | Merkle Hash Tree                                | 默克尔哈希树，用于日志防篡改的树形哈希结构                |
| **MOS**         | Mean Opinion Score                              | 平均意见分，Trneuro 用于评估音质的无参考评分标准          |
| **NMI**         | Non-Maskable Interrupt                          | 不可屏蔽中断，最高优先级硬件中断                          |
| **OOBE**        | Out-of-Box Experience                           | 开箱体验，设备首次开机引导流程                            |
| **OTP**         | One-Time Programmable                           | 一次性可编程存储器，写入后不可修改                        |
| **OUI**         | Organizationally Unique Identifier              | 组织唯一标识符，由 IEEE 分配的 MAC 地址前缀               |
| **PACT**        | Proximity Assistance & Coordinated Transmission | 近域辅助与协同传输协议，本维基核心协议                    |
| **PACT-Cell**   | PACT Battery Cell                               | 独立求生电池，容量 500-1000mAh，专供 Trcomm S1 与 Trsec   |
| **PBL**         | Primary Bootloader                              | 主引导程序，固化于 Trdom ROM，不可刷写                    |
| **PCB**         | Printed Circuit Board                           | 印刷电路板                                                |
| **PCM**         | Pulse Code Modulation                           | 脉冲编码调制，未压缩的数字音频格式                        |
| **PDM**         | Pulse Density Modulation                        | 脉冲密度调制，数字麦克风输出格式                          |
| **PHY**         | Physical Layer                                  | 物理层，OSI 模型第一层                                    |
| **PLR**         | Packet Loss Rate                                | 数据包丢失率                                              |
| **PMIC**        | Power Management IC                             | 电源管理集成电路，Tree 设备中特指 Trpower                 |
| **PUF**         | Physical Unclonable Function                    | 物理不可克隆函数，利用硅片随机差异生成唯一硬件身份        |
| **Ring -1**     | Negative One Privilege Level                    | 最高特权级，Trsec 固件独占                                |
| **RPMB**        | Replay Protected Memory Block                   | 重放保护内存块，用于存储防回滚计数器                      |
| **RSSI**        | Received Signal Strength Indicator              | 接收信号强度指示                                          |
| **RTC**         | Real-Time Clock                                 | 实时时钟，Trsec 内置独立 RTC                              |
| **SBL**         | Secondary Bootloader                            | 二级引导程序，加载 Trsec 并验证 LBT/Leaf OS               |
| **SN**          | Serial Number                                   | 序列号，设备唯一标识                                      |
| **SPI**         | Serial Peripheral Interface                     | 串行外设接口，用于 Trsec 与 Flash 通信                    |
| **SRAM**        | Static Random-Access Memory                     | 静态随机存取存储器，Trneuro 独占 L4 级 SRAM 缓存          |
| **TDFD**        | Trdom Deeply Flash & Download                   | Trdom 深度刷写与下载模式，类比高通 9008                   |
| **TDM**         | Time Division Multiplexing                      | 时分复用，Trcomm S1 主/备实例共享射频时隙                 |
| **TLS**         | Transport Layer Security                        | 传输层安全协议                                            |
| **TMSI**        | Temporary Mobile Subscriber Identity            | 临时移动用户识别码，基站分配的临时身份                    |
| **TRNG**        | True Random Number Generator                    | 真随机数发生器，Trsec 内置硬件随机源                      |
| **TTL**         | Time To Live                                    | 生存时间，PACT 帧中限制最大跳数                           |
| **UID**         | Unique Identifier                               | 唯一标识符                                                |
| **VAD**         | Voice Activity Detection                        | 语音活动检测，用于判断音频流中是否包含人声                |
| **WMM**         | Wi-Fi Multimedia                                | Wi-Fi 多媒体，802.11e 服务质量扩展                        |
| **WP**          | Write Protect                                   | 写保护，eMMC/UFS 物理写保护引脚                           |
