# 快速开始

欢迎使用 Gyoza 博客模板！本指南将帮助你快速搭建自己的博客网站。

## 环境要求

在开始之前，请确保你已经安装以下软件：

- **Node.js** 18.18 或更高版本
- **pnpm** 9.x 或更高版本

## 安装步骤

### 1. 克隆模板

```bash
git clone https://github.com/yourusername/your-blog.git
cd your-blog
```

### 2. 安装依赖

```bash
pnpm install
```

### 3. 启动开发服务器

```bash
pnpm dev
```

现在，打开浏览器访问 `http://localhost:4321`，你就能看到博客了！
