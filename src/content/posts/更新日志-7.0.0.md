# v7.0.0

## 全新框架

1. **新增** 支持 RSS 订阅功能
2. **新增** 支持站点地图生成
3. **新增** 引入 Swup 实现页面过渡动画
4. **新增** 集成 Waline 评论系统
5. **新增** 支持代码高亮与代码块复制功能
6. **重构** 摒弃旧版 HTML 系统，使用 Astro 框架全新重建
7. **重构** 引入 React 组件化开发，提升前端交互体验
8. **重构** 采用 Tailwind CSS 构建样式，支持暗色模式
9. **优化** 使用 pnpm 作为包管理器，提升依赖安装速度
10. **优化** 重构 GitHub Actions 部署流程，支持 pnpm 环境

***

## 界面

1. **新增** Hero 区域展示站点介绍与社交链接
2. **新增** 暗色模式切换功能
3. **新增** 归档页面时间线展示
4. **优化** 响应式布局设计，适配多端设备
5. **优化** 文章页面目录导航与阅读进度条
6. **优化** 项目页面卡片式布局
